# KLSE Stock Analyzer

Retrieve and analyze Kuala Lumpur Stock Exchange (Bursa Malaysia) stock data using Yahoo Finance.

## Setup

```bash
pip install -r requirements.txt
```

## Usage

```bash
python main.py
```

## Features

- Fetch historical stock data for KLSE-listed companies
- Calculate technical indicators (SMA, EMA, RSI, MACD)
- Price trend analysis
- Volume analysis
- Generate visualization charts
